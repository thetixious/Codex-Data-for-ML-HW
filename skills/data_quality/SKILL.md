# Skill: Data Quality

## Goal
Detect at least 3 issue types, offer multiple cleaning strategies, and provide before/after comparison.

## Minimum deliverables
- `reports/quality_report.md`
- `reports/quality_report.html`
- `reports/quality_report.json`
- `reports/quality_comparison.md`
- `reports/quality_comparison.html`
- `reports/strategy_justification.md`
- cleaned dataset in `data/cleaned_dataset.csv`

## Core scripts
- `scripts/detect_issues.py`
- `scripts/fix_data.py`
- `scripts/compare_datasets.py`
- `scripts/save_strategy_justification.py`

## Required interaction
- Show at least 2 cleaning strategy options to user.
- Save user-approved strategy rationale to `strategy_justification.md`.
